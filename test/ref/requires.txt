Required R Packages
====================

- stringr:    for string parsing
- R6:         all distributions are implemented as R6 classes
- extraDistr: for a number of distributions

- chi:     for Chi
- distr:   for Arcsine
- statmod: for InverseGaussian
- skellam: for Skellam
- VGAM:    for Frechet, Levy
